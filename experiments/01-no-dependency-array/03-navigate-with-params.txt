- Calling navigate to the same screen with different params does not trigger animation. It behaves 
like a state change - all 3 callbacks useEffect, useFocusEffect, useLayoutEffect are
"unmounted" and "mounted" each time.

- Calling navigate with params relies on referential stability. If we have a stable object using useMemo 
and call navigate with it, nothing happens. If we generate a new object that is the same, it behaves 
just as if the params were different.

- Navigate(Details) > Navigate(Details) > Back goes back home, since Details was not pushed twice.
